# Handwritten Digit Recognition Using Supervised Learning

## Project Description
This project focuses on pattern recognition using supervised machine learning techniques. The objective is to build a model that can recognize handwritten digits (0–9) using the MNIST dataset.

Algorithms to be implemented:
- k-Nearest Neighbors (KNN)
- Logistic Regression

## Motivation
Handwritten digit recognition is a classic benchmark problem in AI and machine learning. It demonstrates supervised learning and allows comparison of algorithms using measurable results.

## Planned Approach
1. Load MNIST dataset  
2. Preprocess image data  
3. Split into training/testing sets  
4. Train classifiers  
5. Evaluate accuracy and confusion matrix  
6. Analyze successes and failures

## Ground Truth
State-of-the-art models achieve over 99% accuracy using deep neural networks. This project compares simpler models.

## Expected Outcomes
- Working digit classifier
- Algorithm comparison
- Visualization of errors
- Discussion of limitations
